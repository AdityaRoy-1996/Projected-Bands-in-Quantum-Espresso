# Projected-Bands-in-Quantum-Espresso
Projected Electronic Bands in Quantum Espresso

## Requirements
Python
Python Libraries :
  numpy
  sys
  matplotlib
  collections
  scipy
  os
  shutil
Quantum Espresso executables :
  pw.x
  bands.x
  projwfc.x
  
## Notes :
Currently, three types of situations are supported :
  Spin Unpolarized
  Spin Polarized
  Non-Collinear Spin-Orbit
